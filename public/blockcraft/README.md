# Aaria's Block Craft 3D 🐘💙

A gentle, creative-only 3D block-building game made for Aaria — from
**Aaria's Blue Elephant** (https://aariasblueelephant.org) · *Building a New Inclusive World*

## How to run
Just open `index.html` in Chrome or Edge (double-click it). Everything is self-contained
and works offline — no internet, no installs, no accounts.

> 🎤 Voice Mode and read-aloud work best in **Google Chrome**.

## What's inside
- **Creative voxel world** — fly, place and remove colorful blocks (no enemies, no failing, single player)
- **Build Projects** 🏗️ — Cozy Home, Rainbow Car, and a Rocket + Launch Pad with a real countdown blast-off
- **Animal friends** 🐰🦖 — cute pets *and* prehistoric pals wander the world; click them to talk
- **Nilu the Blue Elephant** 💙 — the non-profit's mascot is the in-game guide
- **Expressive communication engine** 💬 — progress moves forward when Aaria picks (or in Voice Mode, *says aloud*) the best full descriptive sentence, not just a single naming word. Gentle coaching, never punishment.
- **Emotions & kindness** 💖 — animals get feelings (sad, hungry, scared…); describing the feeling and choosing a kind action earns hearts and blooms the kindness garden
- **Slime Lab** 🌈 and **Oreo Kitchen** 🍪 — mix, stir, stack and *describe* your creations to unlock slime and cookie blocks
- **Stars & surprises** ⭐ — praise, confetti, surprise animal visitors; everything is read aloud
- **Auto-save** — the world, stars, hearts and unlocks are saved in the browser automatically

## Controls
| Action | Key / Mouse |
|---|---|
| Look around | Click the world, then move the mouse (Esc frees the mouse) |
| Walk | W A S D or arrow keys |
| Fly up / down | Space / Shift |
| Place block | Left click |
| Remove block | Right click |
| Choose block | 1–9 or click the block bar |
| Talk to an animal | Click it |

## Publishing on the website
Upload the whole folder (`index.html`, `js/`, `lib/`) anywhere static —
e.g. `aariasblueelephant.org/blockcraft/` — and link to `index.html`. No server code needed.

## Parent settings ⚙️
In-game Settings lets you toggle: Voice Mode (speech), read-aloud, sounds, music,
and start a fresh world.
